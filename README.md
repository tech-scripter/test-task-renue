EN

Search Engine is intended to search entered value by specified column of a csv file.

Steps to run it:

1. Open CLI

2. Enter commands:
   1) cd path_to_project_dir
   2) mvn package
   3) cd target
   4) java -jar project-0.0.1-SNAPSHOT.jar
